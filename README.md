Utilities
============
This is a space where automation-related codes, scripts, and tools are collected to increase work productivity.

## Setup
- Mac
	```
	// On terminal
	# nano ~/.zshrc
	# export PACKAGE_NAME=com.your.android.package.name // add yout package name
	# source ~/.zshrc
	```
- Linux
	```
	// On terminal
	# nano ~/.bashrc
	# export PACKAGE_NAME=com.your.android.package.name // add yout package name
	# source ~/.bashrc
	```
